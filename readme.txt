A WooCommerce payment gateway for Payler


== Description ==

A WooCommerce payment gateway for Payler offer you to integrate Payler payment gateway in your Wordpress online shopping store.

Payler secure payment system will enabling seamless and reliable payment processing for customers. This plugin allows WooCommerce users to accept payments via Payler, providing flexibility in handling various payment methods like credit cards, direct bank transfers, and other supported options by Payler.

The plugin will include key features like secure encryption for transaction data, compatibility with WooCommerce's order management system, and easy configuration within the WordPress dashboard. It will be designed to meet Payler’s API standards and ensure a smooth, user-friendly checkout experience, with support for payment confirmation, refunds, and error handling directly from the WooCommerce platform.


== Installation ==

1) Direct install - Navigate to "Plugins->Add New" from your Wordpress dashboard, search for 'Payler Payment Gateway', click 'Install Now' and Activate the plugin

2) Uploading ZIP file from Wordpress dashboard - Navigate to the "Plugins->Add New Plugin" in the Wordpress dashboard, Click on 'Upload Plugin' button, select 'payler-payment-gateway.zip' from your computer, click 'Install Now' and Activate the plugin in the Plugin dashboard.

3) Upload plugin ZIP from FTP - Download payler-payment-gateway.zip and extract it, upload the extracted directory to the /wp-content/plugins/ directory and
Activate it from the Plugins dashboard



== Third-Party Services ==

This plugin relies on the following third-party services for its functionality:

1. **Payler Payment Gateway API**
   - **Service URL (Test Mode):** https://facade-api.neo.gate.paylerlab.com/gapi/v1/sessions
   - **Service URL (Live Mode):** https://facade-api.main.gate-api.com/gapi/payout/v1/sessions
   - **Description:** The Payler Payment Gateway is used to process payments through external API requests. When a user initiates a payment through this plugin, the necessary data is sent to Payler’s servers for processing.
   - **Service Terms of Use:** [Payler Terms of Use](https://payler.com/terms-and-conditions-payment-processing)
   - **Privacy Policy:** [Payler Privacy Policy](https://payler.com/privacy_policy)

By using this plugin, you agree that payment data will be transmitted to and processed by Payler as part of the payment process.
